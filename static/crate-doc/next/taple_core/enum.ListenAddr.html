    Memory {
        port: Option<u32>,
    },
    IP4 {

        addr: Option<Ipv6Addr>,
        port: Option<u32>,
    },
}
Expand description

Represents a valid listening address for TAPLE. Internally, they are constituted as a MultiAddr.
